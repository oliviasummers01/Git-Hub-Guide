Step 1: Setting Up the Environment

Install Git and VS Code.

To begin, you must install Git and VS Code. I recommend downloading VS code first as it is much simpler. To install vs code, simply follow the VS Code Download Link and follow the instructions depending on your computer typer. VS code downloads and acts as a normal webapp. Then, to install Git on Mac, first, open the terminal app that can be found in your launchpad. Install Git with the command "xcode-select --install". You will then be prompted to accept the installation and agree to the terms. The download will then take a few minutes to complete. Once the downloading pop up disappears, you can verify that Git has been downloaded by running the command "git" in your terminal which will populate. If it does not populate, restart with the first command.

Configure Git within VS Code (user name, email).

To then use Git within VS Code, you must attach Git to the VS Code webapp. To do this, open VS Code, select View and then Terminal. Once the terminal pannel has open in the bottom half of the VS Code screen, tyle the commands "git config --global user.name "First Last"" and "git config --global user.email name@domain.com". To check to see what has been set in the Git Config File, run the command "git config --global --list".

Step 2: Creating and Cloning a GitHub Repository

Create a new repository on GitHub.

To create your own repository, you must create an account to log into Git Hub. Then, scroll down to the start writing code section. Here you find the option to start a new repository. Simply input a name for your repository, choose whether you'd like it to be public or private (this can be changed later) and select the "create a new repository" button.

Clone the repository using VS Code's integrated Git support.

Once your repository is created, select to copy the link as an HTTPS in the blue quick set up section. Now, in VS Code and open the Command Palette (COMMAND + SHIFT + P). Type the command "> Git: Clone", at this point it will prompt you to enter a repository link. Paste this link to your repository in the command palette. It will then prompt you to create or choose a file to select as the repository destination. Once you've chosen the destination, select the blue button confirming this as your repository destination. It will then ask if you'd like to open the cloned repository, which I would normally do.
